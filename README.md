# 🎓 EduWeb - E-Learning Landing Page

EduWeb is a modern, responsive e-learning landing page designed to showcase online courses, instructors, educational resources, and learning programs. The website focuses on delivering an engaging user experience with a clean UI and responsive design.

## 🌐 Live Demo

🔗 https://minar-eduweb.vercel.app

## 📂 GitHub Repository

🔗 https://github.com/minar-hossain82/eduweb

---

# ✨ Features

- 📚 Modern e-learning landing page
- 📱 Fully responsive design
- 🎓 Course showcase section
- 👨‍🏫 Instructor section
- ⭐ Student testimonials
- 📰 Latest blog section
- 📖 Learning categories
- 🎯 Call-to-action sections
- ⚡ Smooth user experience

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/minar-hossain82/eduweb.git
```

### Go to Project Folder

```bash
cd eduweb
```

### Open Project

Simply open `index.html` in your browser.

Or use VS Code Live Server.
